## Overview

GIGO is a modern, responsive, and interactive professional portfolio website developed using React and Vite.

The website showcases the professional journey, technical expertise, leadership experience, achievements, projects, certifications, and curriculum vitae of an ICT professional currently serving as the Overall Manager at Golf Hotel Kakamega Branch.

The goal is to create a digital identity that goes beyond a traditional portfolio by demonstrating technical competence, leadership, and professional growth.
